#  Overview

AI Global Trading Platform is an intelligent stock market application that provides:

* 📊 Real-Time Stock Market Analysis

* 🧠 AI Market Analyst (LLM)
* 💬 AI Stock Chatbot
* 📈 Technical Indicators
* 🕯 Candlestick Pattern Detection
* 💼 Smart Portfolio Tracker
* 🎓 Stock Market Learning Platform

The application supports:

* 🇮🇳 Indian Stock Market
* 🇺🇸 US Stock Market
* ₿ Cryptocurrency Market

#  Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Global-Trading-Platform.git
```

Go into the project

```bash
cd AI-Global-Trading-Platform
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
