# AI Share Market Platform

An AI-powered stock market analysis and prediction platform.

Features:
- Real-time stock data
- Machine Learning prediction
- LSTM deep learning
- AI chatbot
- News sentiment analysis
- Technical indicators
- Buy/Sell signals
- Risk analysis
- Learning platform

Tech Stack:
- Python
- Streamlit
- TensorFlow
- NLP
- Scikit-learn
- yfinance
