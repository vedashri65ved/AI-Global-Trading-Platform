

# 📈 AI Global Trading Platform

An AI-powered stock market analysis platform that provides real-time stock data, technical analysis, machine learning-based price prediction, trading signals, portfolio tracking, and an intelligent stock assistant chatbot.

---

##  Features

### Real-Time Market Analysis

* Live stock prices using Yahoo Finance API
* Indian Stocks (NSE)
* US Stocks (NASDAQ)
* Cryptocurrency support
* Auto-refresh every 30 seconds

###  Advanced Charts

* Live Price Chart
* Interactive Candlestick Chart
* TradingView Professional Chart
* RSI Indicator Visualization
* MACD Indicator Visualization

###  AI Stock Prediction

* Machine Learning based prediction system
* Random Forest Regressor
* Next-day stock price prediction
* Prediction Confidence Score
* Future Price Forecasting

### Technical Indicators

* RSI (Relative Strength Index)
* MACD (Moving Average Convergence Divergence)
* Bollinger Bands
* Moving Average (MA20, MA50)
* Exponential Moving Average (EMA20, EMA50)
* Volatility Analysis

###  Candlestick Pattern Detection

Detects popular trading patterns:

* Doji
* Hammer
* Bullish Engulfing
* Bearish Engulfing

###  Smart Trading Signals

Generates:

* Strong Buy
* Buy
* Hold
* Sell
* Strong Sell

Based on:

* AI Prediction
* RSI
* MACD
* Moving Average Trend

###  Portfolio Tracker

* Add multiple stocks
* Track investment value
* Profit/Loss calculation
* Portfolio allocation chart
* Buy/Sell recommendations

###  AI Stock Assistant

Powered by Google Gemini AI:

* Stock analysis
* Buy/Sell recommendations
* Risk assessment
* Entry price suggestions
* Target price suggestions
* Beginner-friendly explanations

###  Share Market Learning Center

Learn concepts such as:

* Bull Market

* RSI
* MACD
* Candlestick Charts
* Risk Management

## 📊 Trading Signal Logic

Signal Score is generated using:

| Indicator      | Condition                 |
| -------------- | ------------------------- |
| Prediction     | Future price higher/lower |
| RSI            | Overbought/Oversold       |
| MACD           | Positive/Negative         |
| Moving Average | Bullish/Bearish           |

Final Signals:

* Strong Buy
* Buy
* Hold
* Sell
* Strong Sell
